1️⃣ Einführung in Roblox Lua

Roblox verwendet eine modifizierte Version von Lua 5.1 namens Luau.

Warum Lua für Roblox lernen?

  • ✔ Einfach zu lernen, schwer zu meistern
  • ✔ Wird in Millionen von Spielen verwendet
  • ✔ Hohe Nachfrage nach erfahrenen Roblox-Entwicklern
  • ✔ Kann auf andere Lua-basierte Engines angewendet werden

5️⃣ Skripting Grundlagen

📜 Skript-Typen

Typ Beschreibung Ausführungsort
Script Server-seitige Logik Server
LocalScript Client-seitige Logik Spielergerät
ModuleScript Wiederverwendbarer Code Beide (wenn benötigt)

📡 RemoteEvents & RemoteFunctions

-- Server
local ReplicatedStorage = game:GetService("ReplicatedStorage")
local remoteEvent = ReplicatedStorage:WaitForChild("RemoteEvent")

remoteEvent.OnServerEvent:Connect(function(player, data)
    print(player.Name .. " hat gesendet: " .. data)
end)

-- Client
remoteEvent:FireServer("Hallo vom Client!")

6️⃣ Fortgeschrittenes Skripting

💾 DataStores (Spielerdaten speichern)

local DataStoreService = game:GetService("DataStoreService")
local playerData = DataStoreService:GetDataStore("PlayerData")

game.Players.PlayerAdded:Connect(function(player)
    local data = playerData:GetAsync(player.UserId) or {}
    player.leaderstats.Coins.Value = data.Coins or 0
end)

game.Players.PlayerRemoving:Connect(function(player)
    playerData:SetAsync(player.UserId, {
        Coins = player.leaderstats.Coins.Value
    })
end)

⚡ Optimierungen

  • ✔ Verwende task.spawn() statt spawn()
  • ✔ Cache häufig genutzte Services (local Players = game:GetService("Players"))
  • ✔ Vermeide wait() in Schleifen (nutze task.wait() oder RunService.Heartbeat)

7️⃣ Spielmechaniken (Beispiele)

🏃 Spielerbewegung

local UserInputService = game:GetService("UserInputService")

UserInputService.InputBegan:Connect(function(input, gameProcessed)
    if input.KeyCode == Enum.KeyCode.Space then
        print("Spieler ist gesprungen!")
    end
end)

🔫 Einfaches Kampfsystem

local Players = game:GetService("Players")

local function onPlayerHit(hit, attacker)
    local humanoid = hit.Parent:FindFirstChild("Humanoid")
    if humanoid and attacker then
        humanoid:TakeDamage(10)
    end
end

game.Workspace.Part.Touched:Connect(function(hit)
    onPlayerHit(hit, game.Players:GetPlayerFromCharacter(hit.Parent))
end)

8️⃣ Netzwerke & Sicherheit

🛡️ Anti-Exploit Grundlagen

local function validateToolUse(player, tool)
    if not player.Character or not tool:IsDescendantOf(player.Backpack) then
        return false -- Ungültige Tool-Nutzung
    end
    return true
end

🚫 Remote-Exploits verhindern

-- Immer Client-Anfragen validieren!
remoteEvent.OnServerEvent:Connect(function(player, data)
    if typeof(data) ~= "string" then
        player:Kick("Exploit erkannt!")
        return
    end
end)

9️⃣ Best Practices & Debugging

✅ Do's & ❌ Don'ts

Do:

  • Verwende :WaitForChild() statt direkter Indizierung
  • Organisiere Code mit ModuleScripts
  • Kommentiere deinen Code

Don't:

  • Verwende wait() in kritischen Schleifen
  • Vertraue Client-Daten ohne Validierung
  • Spamme Instance.new() ohne Aufräumen

🐞 Debugging-Tipps

  • Verwende print() für schnelle Checks
  • Roblox Studio Debugger (Breakpoints, Step Through)
  • warn() für wichtige Logs
